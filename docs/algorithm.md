# Algoritmos

Comparativa entre los algoritmos de la lista objetivo y los configs disponibles en el proyecto.

| Algoritmo | Disponible | Nombre en el proyecto |
|---|---|---|
| MLP | Ôťů | `MLP` |
| LightGBM+GFs | ÔŁî | No implementado (requiere framework externo) |
| XGBoost+GFs | ÔŁî | No implementado (requiere framework externo) |
| GatedGCN | Ôťů | `GatedGCN` |
| GAT | ÔŁî | No implementado (hay `GATE`, que es distinto) |
| GIN | Ôťů | `GINE` |
| GIN+RMP | Ôťů | `GINE+RMP` |
| GIN+Ports | Ôťů | `GINE+ports` |
| GIN+Ego ID | Ôťů | `GINE+Ego` |
| GIN+EU | Ôťů | `GINE+EU` |
| PNA | Ôťů | `PNA` |
| PNA+EU | Ôťů | `PNA+EU` |
| Multi-GIN | Ôťů | `Multi-GINE` |
| Multi-GIN+EU | Ôťů | `Multi-GINE+EU` |
| Multi-PNA | Ôťů | `Multi-PNA` |
| Multi-PNA+EU | Ôťů | `Multi-PNA+EU` |
| FraudGT | Ôťů | `SparseNodeGT` |
| FraudGT+RMP | Ôťů | `SparseNodeGT+RMP` |
| FraudGT+Ports | Ôťů | `SparseNodeGT+ports` |
| FraudGT+Ego ID | Ôťů | `SparseNodeGT+Ego` |
| FraudGT+Ports+Ego ID (PE-FraudGT) | Ôťů | `SparseNodeGT+ports+Ego` |
| FraudGT+RMP+Ports+Ego ID (Multi-FraudGT) | Ôťů | `Multi-SparseNodeGT` |

Los configs se encuentran en `configs/AML-Small-HI/` con el prefijo `AML-Small-HI-`.
